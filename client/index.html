Dashboard Loan Default Prediction

About This Project

This capstone project explores machine learning techniques for predicting loan defaults using LendingClub's historical loan data. This model, a Random Forest classifier, analyzes borrower information and loan characteristics to estimate the likelihood of default. The model's prediction functionality is exposed through a Flask API, while visualizations are hosted on Amazon S3 and integrated below to provide insight into feature importance, credit score distributions, and model performance.

The goal is to demonstrate how predictive analytics can support risk assessment in lending, helping financial institutions make data-informed decisions.
